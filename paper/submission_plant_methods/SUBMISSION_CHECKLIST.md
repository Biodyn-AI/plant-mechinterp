## Plant Methods submission-portal checklist

Based on the journal's current author guidelines (plantmethods.biomedcentral.com):

- [ ] **Article type**: Methodology.
- [ ] **Structured abstract**: Background, Results, Conclusions (done in manuscript).
- [ ] **Word count**: Plant Methods has no strict cap for Methodology
  articles; our manuscript is ~7 000 words excluding methods details,
  which is typical.
- [ ] **Keywords**: 4–10 provided at top of manuscript (we list 10).
- [ ] **Abbreviations** list included.
- [ ] **Declarations** section with all six required subsections
  (ethics, consent, data availability, competing interests, funding,
  authors' contributions, acknowledgements).
- [ ] **Data availability**: all code + data accessible at
  <https://github.com/Biodyn-AI/plant-mechinterp> (public); consider also
  depositing the frozen dataset splits and pretrained SAE weights at
  Zenodo and adding a DOI before final acceptance.
- [ ] **Reference format**: Vancouver-style numbered (done: `unsrtnat`
  bibliography style).
- [ ] **Figures**: high-resolution PNGs (300 DPI preferred) — current
  figures are matplotlib 300 DPI.
- [ ] **Tables**: 4 tables, all present inline.
- [ ] **Ethical compliance**: N/A (public genomic data only) — already
  stated in Declarations.
- [ ] **ORCID** for corresponding author — required by BMC.
- [ ] **Institutional-email address** for corresponding author — required.

## Optional supplementary material

Consider preparing these if reviewers request more detail or for
completeness:

- **S1 Dataset summary tables**: per-task class balance, GC statistics,
  chromosome-split breakdown.
- **S2 Full SAE motif-enrichment table**: all 72 significant features
  × top match × q-value × PlantTFDB family, as CSV.
- **S3 Full head-ablation heatmap data**: 12×12 matrix with per-head
  acc_drop + 95% CI.
- **S4 Activation-patching curves**: per-layer heatmaps for canonical-
  base and shuffle corruptions, both directions.

These are generated by the scripts in `analysis/` and exist as JSON/CSV
in `data/real/results/`. Formatting them into Word/PDF supplementary
documents is a quick step before submission if desired.

## Recommended pre-submission sanity checks

1. **Double-check author metadata** in `main.tex` (no placeholder text
   left).
2. **Cover letter** — fill all `*[…]*` placeholders.
3. **Recompile** (`pdflatex main; bibtex main; pdflatex main; pdflatex
   main`) and inspect the final PDF.
4. **Anonymous GitHub link**: the repository is at
   `Biodyn-AI/plant-mechinterp` — make sure it is public / viewable to
   reviewers at submission time.
5. **Zenodo DOI**: optional but recommended — upload a tagged snapshot
   of the repo to Zenodo and add the DOI to the Data Availability
   statement.
6. **Spellcheck and format-check**: run a final pass (aspell / overleaf
   spellcheck) and make sure all abbreviations are defined on first
   use in each section.

## Post-submission notes

- Plant Methods is an open-access journal; an **article-processing charge
  (APC)** applies (~£2 290 / €2 690 / \$2 990 as of recent rates; the
  journal's current APC and any institutional / country waiver policy
  should be verified before submission).
- Expected time to first decision is 30–60 days; typical review process is
  two rounds.
